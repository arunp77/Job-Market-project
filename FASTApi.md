## API Endpoints 

- `/load_data`: Load data into Elasticsearch.
- `/usecase1`: Execute and retrieve results for use case 1 (e.g., analyzing job postings).
- `/usecase2`: Execute and retrieve results for use case 2 (e.g., tracking market trends).
